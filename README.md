# Koala Wellness & Spa — Demo Site

One-page marketing website for **Koala Wellness & Spa**, a massage and wellness spa in Kirkstall VIC, Australia.

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- Mobile-first, Vercel-ready

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Import the repo into Vercel — no environment variables required.
